# 🌐 Portfolio Website

Struktur project website portofolio pribadi.

## 📁 Struktur Folder

```
portfolio/
├── index.html              ← File utama HTML
├── README.md
│
├── css/
│   ├── style.css           ← Styling utama
│   ├── responsive.css      ← Media queries (mobile/tablet)
│   └── animations.css      ← Animasi & transisi
│
├── js/
│   ├── main.js             ← Logic utama & navbar
│   ├── animations.js       ← Animasi scroll & efek
│   └── contact.js          ← Form kontak
│
├── sections/               ← Bagian-bagian halaman
│   ├── hero.html           ← Section pembuka / landing
│   ├── about.html          ← Tentang saya
│   ├── skills.html         ← Keahlian / skills
│   ├── projects.html       ← Daftar proyek
│   └── contact.html        ← Kontak
│
├── components/             ← Komponen yang dipakai ulang
│   ├── navbar.html         ← Navigation bar
│   ├── footer.html         ← Footer
│   └── project-card.html   ← Card template proyek
│
└── assets/
    ├── images/
    │   ├── profile/        ← Foto profil
    │   ├── projects/       ← Screenshot proyek
    │   └── icons/          ← Ikon & logo
    ├── fonts/              ← Custom fonts (jika ada)
    └── videos/             ← Video background (jika ada)
```
